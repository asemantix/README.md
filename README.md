# Package contents

This bundle contains the patent-supporting artifacts: reports, CSVs, math appendix, and a machine-readable manifest.

## Files
- replication_manifest.md — human-readable artifact list.
- manifest.json — machine-readable manifest with SHA-256 checksums.
- math_appendix.md — math appendix.
- ablations_sensitivities.csv — sensitivity sample.
- ablations_report.html — sensitivity report.
- ablations_extended.csv — extended grid.
- ablations_extended_report.html — extended report.

## How to verify integrity
Use the sha256 values in manifest.json to verify downloads.

## How to view
Open the HTML files in a browser; CSVs in any spreadsheet; MD in any editor.
